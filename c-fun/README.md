## Overview

This project focuses on understanding how high-level code translates into machine-level instructions and practicing reverse engineering and debugging compiled binaries. Designed for Linux systems, the target architecture here is x86_64 (as verified using uname -m).

#### Requirements

- Basic C programming skills
- Familiarity with assembly (specific to your architecture, e.g., x86_64)
- Linux
- Determination

#### Goalz

- Reverse engineer and debug binaries
- Explore how high-level code maps to assembly and machine instructions

Start small, dig deep, and keep learning.
