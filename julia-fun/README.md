#### Pluto.jl on localhost

⇧+Enter:   run cell 

Ctrl+Enter:   run cell and add cell below

Ctrl+S:   submit all changes

Delete or Backspace:   delete empty cell

PageUp or fn+↑:   jump to cell above

PageDown or fn+↓:   jump to cell below

Ctrl+click:   jump to definition

Alt+↑:   move line/cell up

Alt+↓:   move line/cell down


Ctrl+/:   toggle comment

Ctrl+M:   toggle markdown

Ctrl+Shift+[:   hide cell code

Ctrl+Shift+]:   show cell code

Ctrl+Q:   interrupt notebook


Select multiple cells by dragging a selection box from the space between cells.

Ctrl+C:   copy selected cells

Ctrl+X:   cut selected cells

Ctrl+V:   paste selected cells


The notebook file saves every time you run a cell
